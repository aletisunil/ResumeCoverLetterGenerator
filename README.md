# ResumeCoverLetterGenerator

### AI Career Agent Team
A multi-agent application powered by LangGraph and Streamlit that autonomously tailors your resume and writes a cover letter based on a specific Job Description (JD).

#### Key Features
PDF Support: Upload your existing resume directly.

Intelligent Analysis: An Evaluator Agent scores your resume against the JD and identifies missing keywords.

Dual Modes: Choose between "Slight Polish" (grammar/keywords) or "Aggressive Rewrite" (restructuring bullet points).

Parallel Execution: The Resume Writer and Cover Letter Writer agents work simultaneously to save time.
